Design System Frontend Project

This project is a modern design system UI built using Next.js 15 and Tailwind CSS v3. It showcases structured design principles including colors, typography, icons, and reusable components.

⚙️ Installation & Setup

Follow these steps to run the project locally:

1️⃣ Clone the repository
git clone <your-repo-link>
cd <your-project-folder>
2️⃣ Install dependencies

Make sure you are using Next.js 15 and Tailwind CSS v3

npm install
3️⃣ Install icon package
npm install lucide-react
4️⃣ Run the development server
npm run dev
